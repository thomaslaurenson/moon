# Python Badge Row

```markdown
![Python Version](https://img.shields.io/pypi/pyversions/{package}?logo=python) ![Code Coverage](https://img.shields.io/badge/Coverage-XX%25-blue?logo=python)
```

Python version comes from PyPI. If the project is not on PyPI, use a static badge instead: `![Python Version](https://img.shields.io/badge/python-3.x%2B-blue?logo=python)`. Replace `XX` with the coverage percentage on each release.
